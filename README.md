# Upakuem.pro — Telegram-бот калькулятор КП

Telegram-бот для расчёта стоимости фулфилмент-услуг компании **Upakuem.pro**. Клиент описывает заказ текстом, бот через LLM (Google Gemini 2.0 Flash via OpenRouter) разбирает запрос, подбирает услуги из прайс-листа и возвращает готовое коммерческое предложение со скидкой 20% на первый заказ.

## Возможности

- Команда `/start` — приветствие и сброс сессии
- Команда `/price` — краткий прайс-лист по основным категориям
- Команда `/help` — инструкция по использованию
- Свободный текстовый запрос → готовое КП в Telegram
- Уточняющие вопросы при неполном описании заказа
- Скидка 20% применяется автоматически

## Быстрый старт

```bash
# 1. Установить зависимости
npm install

# 2. Создать .env на основе примера
cp .env.example .env

# 3. Заполнить переменные в .env
# TELEGRAM_BOT_TOKEN — токен от @BotFather
# OPENROUTER_API_KEY — ключ от openrouter.ai

# 4. Запустить бота
npm start
```

## Переменные окружения

| Переменная | Описание |
|---|---|
| `TELEGRAM_BOT_TOKEN` | Токен Telegram-бота (получить у @BotFather) |
| `OPENROUTER_API_KEY` | API-ключ OpenRouter (openrouter.ai) |

## Структура проекта

```
src/
├── index.js   # Точка входа, инициализация бота и обработчики
├── llm.js     # Обёртка над OpenRouter API
├── price.js   # Прайс-лист и коэффициент скидки
└── prompt.js  # Системный промт для LLM
```
